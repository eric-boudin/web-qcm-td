
# Exercice IUT

IUT Sophia-Antipolis - DUT 2018/2019 - Web

### Installation

 - [ ] Require [Composer](https ://getcomposer.org/) to run.

```sh
$ composer install
```

### Run

On Linux/Mac:
```sh
$ ./vendor/bin/phpunit --testdox tests
```

On Windows:
```batch
> vendor\\bin\\phpunit --testdox tests
```
